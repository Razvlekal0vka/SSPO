                                        <-RoboWorldOne->


    Наш проект - игра, основана на библиотеке pygame, также соответсвующая стандартам довольно известного
жанра игр - Рогалик. В игре будут реализованны алгоритмы процедурной генерации карты мира, создание
всевозможных объектов с разнообразными свойствами(аванпосты, враги, сундуки и тп), адаптивные звуки и многое другое.





                            Микро сюжет нашей игры(для понимания)

                    По сюжету вы оказываетесь в сказочном мрачном мире,
                    где вы должны сделать свой выбор: стать героем и
                    спасителем этого мира или найти способ выбраться из
                    матрицы и вернуться к обычной жизни.




                                ОПИСАНИЕ СОДЕРЖАНИЯ ПРОЕКТА

Папки:
    data - стартовая папка с текстур-паками и текстовыми файлами, созданных для хранения информации, которая
           предназначена для *иных программ*.

    test_data - самая актуальная папка с текстур-паками и новыми текстовыми файлами, созданных для хранения необходимой
    информации.

    *Более подробное описание файлов папок появится позже*

Программы:
    Программы типа test_MAP_GENERATION - разновидные алгоритмы генерации карты

    Программы типа test_version_of_MAIN_GAME - основные тестовые версии игры, в которых тестируются крупные нововведения

    MAIN_GAME - самая актуальная версия игры(в нее уже встроена самая актуальная версия генерации карты)

    MAIN_MAP_GENERATOR - самая актуальная версия генерации карты

    Остальные:
        *Иные программы* - не столь значимые программы. В них тестировались определенные нововведения, которые либо уже
        присутствуют в MAIN_GAME, либо являются более сырыми версиями MAIN_GAME.

Задачи.txt - здесь записаны все наши ближайшие цели и нововведения
